class jack\+::channel$<$ sig\+\_\+tp $>$}
Many-\/to-\/one synchronization object. Enable a consumer to wait for different signal types and handle each accordingly. 

A waiter is determined to be wake-\/eligible if a bitwise-\/\+AND between the sent signal and the waited signal returns a non-\/zero value.
