\doxysubsection{Detailed Description}
A thread synchronization object that garauntees one thread is woken per \mbox{\hyperlink{classjack_1_1sticky__event_a3c0c75de1ca26c0513fafb2f138027f3}{set()}} call. This means that if no threads are waiting when \mbox{\hyperlink{classjack_1_1sticky__event_a3c0c75de1ca26c0513fafb2f138027f3}{set()}} is called, the next wait will return immediately. 

\begin{DoxyNote}{Note}
Calling \mbox{\hyperlink{classjack_1_1sticky__event_a3c0c75de1ca26c0513fafb2f138027f3}{set()}} multiple times between waiters will have no additional effect. 
\end{DoxyNote}
